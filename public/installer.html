Installation Details

This installer will set up a dedicated environment for Gemini CLI specifically for OpenClaw.

Dedicated Installation:

Gemini CLI is installed in isolation within this folder (src/.gemini).

No Global Interference:

If you already have Gemini CLI installed on your PC, it will not be affected. Settings and credentials are not shared.

Important Warning

This software enables "YOLO Mode" by default.

Gemini CLI will attempt to execute commands, create, edit, or delete files automatically without asking for permission to fulfill your AI requests.

  • Do NOT run this on production servers or directories containing sensitive unbacked-up data.
  • Only use this within safe, tested environments.
